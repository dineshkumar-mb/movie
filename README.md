# movie-api

provide a movie api for react js app.

1.   `npm install`
2.    `npm run server`
3.   `git init`
 4.  ` git add`
 5.  ` git commit - "Your MSG"`

 6.  ` git push-u origin main or git push`



#Database options

2. `NOSQL - (MongoDb)`
